\# Vision



\## O problema



Gerenciar compras domésticas normalmente depende da memória.



É comum esquecer produtos importantes, comprar itens em excesso ou não perceber mudanças nos hábitos de consumo.



Aplicativos de lista de compras geralmente são manuais e não aprendem com o comportamento dos usuários.



Por outro lado, sistemas completos de controle de estoque exigem muito esforço para serem mantidos, tornando-se inviáveis para o uso diário.



\---



\## A solução



Desenvolver um assistente inteligente de abastecimento doméstico.



O sistema deve aprender os hábitos dos moradores, manter uma estimativa confiável da despensa e gerar recomendações úteis para facilitar as compras.



O foco é reduzir o esforço do usuário, não controlar estoque com precisão absoluta.



\---



\## Público-alvo



Famílias ou pessoas que realizam compras recorrentes para casa.



Inicialmente o sistema será utilizado por uma única residência compartilhada entre seus moradores.



\---



\## Objetivos



\- Simplificar o gerenciamento das compras.

\- Evitar esquecimentos.

\- Reduzir desperdícios.

\- Aprender padrões de consumo.

\- Tornar as recomendações melhores conforme o uso aumenta.



\---



\## Filosofia



O aplicativo deve funcionar como um assistente.



O usuário não deve precisar alimentar constantemente o sistema para obter valor.



Quanto mais o sistema aprende, menos esforço deve ser exigido.



\---



\## Visão de longo prazo



O sistema deve evoluir para responder perguntas como:



\- O que devo comprar hoje?

\- Quanto devo comprar?

\- O que está acabando?

\- Meu consumo mudou?

\- Estou gastando mais do que normalmente gasto?

\- Vale a pena aproveitar essa promoção?



O objetivo é se tornar um assistente inteligente para abastecimento doméstico.
